# Forex-Bot-2-bars
Simple forex trading bot with clean strategy and ready to run

Just clone this repo and run on cron everyday at 00:01! 

print('api-sell-command') - replace for sell command from your broker.
print('api-buy-command') - replace for buy command from your broker.

Tradeng strategy:
If previous two days has the same trend - we open position to those trend.

(You need api key from https://fcsapi.com/ to check the market prices.)
